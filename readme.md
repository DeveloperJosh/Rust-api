# Rust REST API

This is a RESTful API built with Rust, designed to handle posts, likes, and user authentication features including logins and sign-ups. The service uses MongoDB for efficient data storage and retrieval, supporting scalability as user interactions increase.

## Features

- **Posts Management**: Create, read, update, and delete posts (coming soon).
- **Likes**: Users can like posts.
- **User Authentication**: Support for user logins and sign-ups (coming soon).

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Before you begin, ensure you have MongoDB installed and running on your system. Follow the MongoDB installation guide suitable for your operating system: [MongoDB Installation](https://docs.mongodb.com/manual/installation/).

You also need Rust installed:

```bash
rustup update
rustup install stable
cargo run
```
